AI-powered article generator for WordPress that helps you create high-quality blog posts quickly.

== Description ==

WordPress Article Generator is a powerful plugin that helps you generate high-quality articles for your WordPress site. With customizable options for article length, tone, and keywords, you can create content that matches your site's style and SEO requirements.

Features:

* Generate articles based on title and keywords
* Choose article length (300-1500+ words)
* Select writing tone (Professional, Casual, Informative, Persuasive)
* Preview generated articles before saving
* Save as draft or publish immediately
* Automatic tag generation
* Clean and intuitive admin interface

== Installation ==

1. Upload the `wordpress-article-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Article Generator' in the WordPress admin menu
4. Start generating articles!

== Frequently Asked Questions ==

= How does the article generation work? =

The plugin uses AI technology to generate unique articles based on your input. You provide a title, keywords, desired length, and tone, and the plugin creates relevant content.

= Can I edit the generated articles? =

Yes! All generated articles can be edited before saving. You can modify the title, content, excerpt, and tags.

= What AI service does this use? =

This is a demonstration plugin that generates sample content. For production use, you would integrate with AI services like OpenAI, Claude, or others.
